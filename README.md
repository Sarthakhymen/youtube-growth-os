# 🚀 YouTube Growth OS

An ultra-premium, AI-powered content factory designed to help creators scale from 0 to 1M+ subscribers. Built with **Next.js 16**, **Razorpay**, and **NVIDIA NIM AI**.

## ✨ Features

- **Viral Idea Generator**: Data-driven video concepts optimized for the YouTube algorithm.
- **Retention-Optimized Scripts**: AI-generated scripts with built-in hooks and pattern interrupts.
- **Shorts Factory**: Rapidly convert long-form ideas into high-impact vertical content.
- **Channel Analytics AI**: Advanced audits and actionable growth steps.
- **Premium Design**: Obsidian Lumina design system with glassmorphism and fluid animations.
- **Secure Payments**: Integrated Razorpay checkout flow for Pro upgrades.

## 🛠️ Tech Stack

- **Framework**: [Next.js 16 (App Router)](https://nextjs.org/)
- **Styling**: Vanilla CSS Modules (Premium Aesthetics)
- **Icons**: Lucide React
- **AI Engine**: NVIDIA NIM (Qwen-3-Coder)
- **Payments**: Razorpay SDK
- **Database**: MongoDB (via Mongoose)
- **Authentication**: Clerk

## 🚀 Getting Started

### 1. Prerequisites

- Node.js 18.x or higher
- A Razorpay account (for payments)
- An NVIDIA NIM API key (for AI features)

### 2. Installation

```bash
git clone https://github.com/YOUR_USERNAME/youtube-growth-os.git
cd youtube-growth-os
npm install
```

### 3. Environment Setup

Create a `.env.local` file in the root directory and add your keys:

```env
# AI (NVIDIA NIM)
NVIDIA_API_KEY=your_key_here

# Payments (Razorpay)
RAZORPAY_KEY_ID=your_key_id
RAZORPAY_KEY_SECRET=your_key_secret
NEXT_PUBLIC_RAZORPAY_KEY_ID=your_key_id

# Database
MONGODB_URI=your_mongodb_uri
```

### 4. Run Locally

```bash
npm run dev
```
